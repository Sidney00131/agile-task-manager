# Agile Task Manager

Sistema simples de gerenciamento de tarefas desenvolvido em Python, utilizando metodologia ágil Kanban e controle de versionamento com GitHub.

## Objetivo
Permitir o cadastro, acompanhamento e controle de tarefas, simulando o fluxo de trabalho de um time ágil.

## Metodologia
Foi adotado o método Kanban, utilizando o GitHub Projects para organizar as tarefas nas colunas To Do, In Progress e Done.

## Tecnologias
- Python
- PyTest
- GitHub Actions
