# Agile Task Manager

Sistema simples de gerenciamento de tarefas desenvolvido em Python, com o objetivo de simular o uso de metodologias ágeis no desenvolvimento de software, utilizando o GitHub como ferramenta principal de gestão, versionamento e controle de qualidade.

## Objetivo do Projeto
O objetivo deste projeto é permitir o cadastro, acompanhamento, atualização e remoção de tarefas (CRUD), possibilitando a visualização do fluxo de trabalho de um time ágil por meio de um quadro Kanban no GitHub Projects.

O projeto foi desenvolvido como atividade prática da disciplina de Engenharia de Software, integrando conceitos de planejamento, versionamento, testes automatizados e gestão de mudanças.

## Escopo Inicial
Inicialmente, o sistema contemplava apenas as seguintes funcionalidades:
- Criação de tarefas
- Listagem de tarefas
- Atualização de tarefas
- Remoção de tarefas

As tarefas possuem título, descrição e status de conclusão.

## Metodologia Ágil Utilizada
Foi adotada a metodologia **Kanban**, utilizando o recurso **GitHub Projects**, com as seguintes colunas:
- To Do
- In Progress
- Done

Essa abordagem permitiu acompanhar visualmente o andamento das tarefas, melhorar a organização do trabalho e facilitar o controle das entregas ao longo do desenvolvimento.

## Tecnologias Utilizadas
- Python 3
- PyTest (testes automatizados)
- GitHub Actions (integração contínua)
- GitHub Projects (Kanban)
- GitHub (controle de versão)
